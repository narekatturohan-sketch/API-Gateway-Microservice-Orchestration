# API Gateway + Microservice Orchestration

## 📌 Overview
This project implements a lightweight API Gateway using FastAPI.  
The gateway routes requests to multiple microservices (Validation, Rule Engine, ETL, Automation), demonstrating orchestration and scalable architecture design.

---

## 🛠️ Tech Stack
- **Language:** Python 3.14
- **Framework:** FastAPI
- **Libraries:** requests, uvicorn
- **Tools:** Git, Postman, Docker (optional)
